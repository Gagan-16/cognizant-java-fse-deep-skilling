Objective

Develop a financial forecasting tool that predicts future values using a recursive algorithm based on a fixed annual growth rate.

Understanding Recursion

Recursion is a programming technique in which a method calls itself to solve a smaller version of the same problem.

A recursive solution consists of:

Base Case: Stops the recursion.
Recursive Case: Calls the method again with a reduced problem size.


Time Complexity Analysis
Recursive Solution
Time Complexity: O(n)
Space Complexity: O(n)

The method performs one recursive call for each year of forecasting
