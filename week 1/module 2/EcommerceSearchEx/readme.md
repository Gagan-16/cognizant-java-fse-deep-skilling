Objective

Implement Linear Search and Binary Search algorithms to search products in an e-commerce platform and compare their performance.

Big O Notation

Big O notation is used to describe the efficiency of an algorithm by measuring how its execution time grows as the input size increases.

Search Operation Scenarios
Linear Search
Best Case: O(1)
Average Case: O(n)
Worst Case: O(n)

Binary Search
Best Case: O(1)
Average Case: O(log n)
Worst Case: O(log n)
